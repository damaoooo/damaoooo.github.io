Windows

windows下想用pintools真的非常困难了

首先还是下载源码包，修改源码，最难的是编译，因为windows下面的make，真的很难用

首先是要安装visual studio， 然后就是安装cygwin的make，最后添加环境之后，make还是不成功。

硬了，拳头硬了，而且看其他的博客说windows上面，即便是输入相同，输出的指令也不同，不得不说在这方面linux还是要好得多

弃坑不搞了

angr

安装pip install angr
